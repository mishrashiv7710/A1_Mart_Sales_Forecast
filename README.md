# 📊 A1 Mart Sales Forecasting

**🌐 Live Demo:** [https://a1-mart-sales.streamlit.app/](https://a1-mart-sales.streamlit.app/)

An interactive dashboard and sales forecasting tool built using **Streamlit** and **Machine Learning (Random Forest)**.

## 🚀 Key Features

- **🔐 User Authentication**: Secure login and signup functionality.
- **📊 Real-time Dashboard**: Track total sales, profit, and quantity across multiple categories.
- **📈 Advanced Visualization**: Dynamic pie charts and monthly bar charts for category distribution.
- **🔮 AI Sales Forecast**: Predict future sales based on historical data using Random Forest Regression.
- **📁 Ready for Deployment**: Perfectly configured for Streamlit Community Cloud and Vercel.

---

## 🛠️ Tech Stack

- **Python**: Core logic and backend.
- **Streamlit**: Interactive user interface.
- **Pandas & NumPy**: Efficient data processing.
- **Plotly**: Stunning, interactive visualizations.
- **Scikit-learn**: Machine learning model training.

## 🏃 How to Run Locally

1. **Clone the repository**:
   ```bash
   git clone https://github.com/mishrashiv7710/A1_Mart_Sales_Forecast.git
   cd A1_Mart_Sales_Forecast
   ```

2. **Create a virtual environment**:
   ```bash
   python -m venv .venv
   source .venv/bin/activate  # Windows: .venv\Scripts\activate
   ```

3. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

4. **Run the app**:
   ```bash
   streamlit run app.py
   ```
